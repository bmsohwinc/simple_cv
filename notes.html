Detailed Short notes

For HTML

UTF-8

- UTF-8 is the widely used encoding, widely rendered presently.

- Here is a link for the UTF-8 table

<em> and <i>

- Both italicize the word, but em is for stressing that word, but i is to just stylize that word as an italic

Listing

- ul means unordered listing - ol means ordered listing - li specifies listing the element
- start and type can be used to change the listing format

Images

- Pretty straight forward, height, width can be used

Download

- Works if it is unable to display in browser
- If it is a PDF, or some file that can be easily rendered by the browser, then it navigates to that link rather than downloading
- Read this for better understanding.

Tables

- table for making space one table
- tr for making the table row
- td for entering table data (a column)
- thead for heading of column
- tfoot and tbody for footer and body resp.
- th to enter Heading data

Forms

- Most widely used for dynamic contents. Always associated with JS
- Elements occur as Label:Input pairs
- Labels are like indicators; Inputs are like variables holding values
- action specifies whether to goto some page, or mail someone (mailto:...), etc
- method is nothing but the HTTP GET/POST/DELETE methods
